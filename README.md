## 📦 Instalación

### Requisitos previos
- Node.js (v14 o superior)
- npm o yarn

### Pasos de instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/juanda7426/Juanda-Code.git
   cd Juanda-Code
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Configura las variables de entorno:
   - Copia `.env.local` (si existe) o crea uno
   - Establece tu `GEMINI_API_KEY`:
   ```
   GEMINI_API_KEY=tu_clave_api_aqui
   ```

4. Ejecuta el proyecto localmente:
   ```bash
   npm run dev
   ```

5. Abre tu navegador en `http://localhost:3000` (o el puerto especificado)
